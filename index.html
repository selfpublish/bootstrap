Bootstrap 4 Grid System Previewer

Please note, this tool is meant for previewing the layout for Bootstrap grid columns. While it does support the "xs" column specification, this is a carry-over from Bootstrap 3 and is no longer required. It now supports Bootstrap's "mx-auto", "mr-auto", and "ml-auto" for positioning, although grid-system support for these classes (e.g. "ml-md-auto") has not yet been enabled.

You should see colored boxes representing the layout of the given divs in the respective screen size categories below. If any or all of the boxes remain empty, you have entered an invalid column class.

As an example, try entering "col-12 col-md-6 col-xl-3" in the first input field
and "col-12 col-sm-10 col-md-8 col-lg-6 col-xl-4" in the second.
